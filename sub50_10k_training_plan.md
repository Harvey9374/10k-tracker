# Sub-50 Minute 10km Training Plan
## Age 42 | Start: May 2026 | Target Race: Mid-May 2027

---

## Objective

Drop from a ~59-61 min flat equivalent to sub-50 min over 52 weeks. That is 5:00/km average pace. You have done 42-45 min before, so the physiological ceiling is not the problem. Consistency and structured load is.

---

## Equipment

- 2 x 15kg dumbbells
- Ropeless skipping rope
- Outdoor running

---

## Weekly Schedule Framework

| Day | Availability | Session Type |
|---|---|---|
| Monday | 90 mins | Key run + optional strength |
| Tuesday | 90 mins | Strength + conditioning |
| Wednesday | Limited, home | Skip + mobility or light strength |
| Thursday | Limited, home | Mobility or light strength |
| Friday | 60 mins | Run session |
| Weekend (alt) | Variable | Bonus long run |

---

## Pace Guide (These evolve across phases)

| Session Type | Phase 1 | Phase 2 | Phase 3 | Phase 4 |
|---|---|---|---|---|
| Easy run | 6:30-7:00/km | 6:15-6:45/km | 6:00-6:30/km | 6:00-6:30/km |
| Steady/tempo | N/A | 5:40-5:55/km | 5:20-5:35/km | 5:05-5:20/km |
| Interval reps | N/A | 5:10-5:25/km | 4:55-5:10/km | 4:45-5:00/km |
| Race pace | N/A | N/A | 5:05/km | 5:00/km |

Easy runs should feel genuinely easy. You should be able to hold a full conversation. If you are breathing hard, slow down.
